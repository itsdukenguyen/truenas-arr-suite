# TrueNAS SCALE *arr Suite Setup

Complete documentation for my automated media server stack on TrueNAS SCALE:

- **Radarr** - Movies
- **Sonarr** - TV Shows
- **Lidarr** - Music
- **Bazarr** - Subtitles
- **Transmission** (on Raspberry Pi) + custom move script
- **Jellyfin** - Media playback
